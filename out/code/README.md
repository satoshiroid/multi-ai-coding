# Setup and Run

## Prerequisites
- Docker
- Docker Compose

## Setup
1. Copy `.env.example` to `.env` and configure your environment variables.

## Run
1. Execute `docker-compose up --build` to start the services.
2. Access the application at `http://localhost:3000`.

## Test
1. Upload two sample PPTX files via the web interface.
2. Verify that the files are compared and displayed correctly.
